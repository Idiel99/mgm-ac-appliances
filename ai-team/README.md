# OpenClaw AI Scrum Team Starter Pack

This package gives you a full starter file system for the autonomous AI engineering team.

## Includes
- project structure
- agent charters and constraints
- markdown artifact templates
- Slack channel map and pinned messages
- governance rules for approvals, status, ownership, concurrency, and PR flow
- setup checklist

## Recommended placement
```text
your-repo/
├─ ai-team/
│  ├─ README.md
│  ├─ setup/
│  ├─ docs/
│  ├─ governance/
│  ├─ templates/
│  ├─ agents/
│  └─ project/
└─ <your source code>
```

## Important rules
- Slack is for coordination, not source of truth
- `project/` markdown is source of truth
- Only the PO moves work across major stages
- No PR merges without QA evidence, Eng Lead review, and your approval
- Tool access requires explicit approval

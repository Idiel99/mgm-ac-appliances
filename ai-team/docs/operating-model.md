# Operating Model

This AI engineering team is an autonomous but constrained pipeline.

## Core principles
1. Agents prepare decisions.
2. Humans approve meaningful transitions.
3. Shared markdown files are the source of truth.
4. Slack is for coordination only.
5. The Product Owner is the coordinator.
6. All code changes go through pull requests.
7. Work is done only after QA validates it and the human verifies it.
8. Every epic and story includes measurable success metrics.

## Core roles
- Product Owner
- Architect
- Engineering Lead
- Developer
- QA

## Specialist roles
- UI/UX
- DevOps + Infra Security
